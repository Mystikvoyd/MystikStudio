# _TEMPLATE — How to add a new tool
# ================================
# 1. Copy this folder into Creators/ (or webpage/)
# 2. Rename it to your tool name (e.g. "my-image-scaler")
# 3. Edit tool.json — set name, description, color, launcher
# 4. Edit launcher.vbs — point to your actual script
# 5. Edit run.ps1 — your tool's PowerShell logic
# 6. Restart the MystikStudio Dashboard — button appears automatically
