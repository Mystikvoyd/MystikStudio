# Story Dashboard App

This is the editable desktop version of the local story dashboard.

Double-click:

```text
story-dashboard-app\Open Story Dashboard App.cmd
```

It reads the same local files as the webpage:

```text
book\manuscript
book\reference
book\story-dashboard.config.json
```

The app tracks manuscript words, session progress, goals, chapter files, and the Project Codex reference package. Its source is plain PowerShell:

```text
story-dashboard-app\Start-StoryDashboardApp.ps1
```

Everything stays local to this workspace.
